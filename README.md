# 🛡️ Threat Detection System

## 📌 Overview

The Threat Detection System is a cybersecurity-focused application designed to identify and analyze different types of online threats.
It detects harmful activities such as phishing, fake news, cyberbullying, and network-based attacks using intelligent techniques.

---

## 🚀 Features

* 🔍 Phishing detection
* 📰 Fake news detection
* 💬 Cyberbullying analysis
* 🌐 Network security monitoring
* 📊 Threat classification and reporting
